a partir do  kernel 2.6 o linux apresenta o sistema de arquivo ~sysfs~, este file system é descrito como a união dos sistemas de arquivo ~proc~, ~devpty~ e ~devpty~.

o system file ~sysfs~ é montado em /sys/ e contém diretórios que organizam os dispositivos conectados ao sistema de diversas maneiras.

o dir dev/ armazena informações referentes a arquivos da memoria ram, arquivos que identificam HDs, mouse, IDEs, TTYs

o dir proc/ é composto por dirs que na verdade são nomeados com o PID de um processo especifico.

* identificar dispositivos IDE, SATA e SCSI
** SCSI
todos os dispositivos SCSI que o sistema encontra são listados em ~/proc/scsi/scsi~, eles possuem 2 tipos principais ->

- 8 bits que concta até 7 dispositivos

- 16 bits que conecta até 15 dispositivos

** IDE
os dispositivos de disco IDE são representados como ~/dev/hd*~, sendo o primeiro disco ~/dev/hda~ e o secundário ~/dev/hdb~

** SATA
os dispositivos SATA são representados sempre como ~/dev/sd*~, sendo o primeiro disco ~/dev/sda~ e o secundário sendo representado como ~/dev/sdb~
